java 문자열에서 숫자만 추출

JAVA
2020-12-17 20:55:14
이회구
# java 문자열에서 숫자만 추출 ```java //정규표현식으로 Integer 추출 String str = "aaa1234, ^&*2233pp"; //replaceAll()에 인자로 정규표현식과 변환할 문자열을 전달 String intStr = str.replaceAll("[^0-9]", ""); //"[^0-9]"는 0~9의 숫자가 아닌 문자열을 의미 (숫자가 아닌 문자들을 공백("")) System.out.println(intStr); // 출력값: 12342233 //replaceAll2탄 String str = "aaa1234, ^&*2233pp"; //"^\\d"는 "[^0-9]"의 축약형으로 의미가 동일("[^0-9]" 대신 "^\\d"사용) String intStr = str.replaceAll("[^\\d]", ""); System.out.println(intStr); // 출력값: 12342233 //for문으로 숫자 추출 String str = "aaa1234, ^&*2233pp"; String intStr = ""; for (int i = 0; i < str.length(); i++) { char ch = str.charAt(i); // 48은 ASCII에서 숫자 0을 의미하고, 57은 ASCII에서 숫자 9를 의미(0~9까지 숫자만 추출) if (48 <= ch && ch <= 57) { intStr += ch; } } System.out.println(intStr); // 출력값: 12342233 ```
